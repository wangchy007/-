# 主动安全与被动安全
车辆安全中主动安全与被动安全的基本概念

这句话介绍了汽车的被动安全系统 (Passive Safety Systems)，其核心是在碰撞发生后保护乘客，将伤害降至最低。

💡 核心概念解析
•   被动安全系统 (Passive Safety Systems)
    与旨在避免事故的主动安全系统（如ABS、ESP）不同，被动安全是在事故已发生后起作用的“最后一道防线”，主要包括安全带、安全气囊、吸能车身等 。
•   碰撞保护 (Crash Protection)
    指在碰撞过程中及之后，通过一系列设计来保护车内乘员，减轻其受到的伤害 。
🚗 作用原理

当车辆发生碰撞时，被动安全系统会通过以下方式协同工作：

•   约束乘员：安全带将人固定在座椅上，防止其因惯性前冲或飞出车外。

•   缓冲撞击：安全气囊在瞬间弹出，在人与方向盘、挡风玻璃等硬物之间形成缓冲层。

•   吸收能量：车身的溃缩吸能区会变形，吸收部分撞击能量，从而减少传递到乘员舱的冲击力 。

简单来说，“溃缩区 (Crumple Zones)”是车身前后专门设计的“可变形”区域。在碰撞时，它们会像“牺牲自己”一样发生折叠、压扁，以此来吸收撞击产生的巨大能量。

这样做的好处是，能将原本会直接冲击到驾驶舱和乘客身上的能量大幅削弱，从而有效保护车内人员的安全。

🛡️ 防止撞击 (Protect occupants from hitting interior surfaces)
在车辆发生碰撞时，人体会因惯性猛烈前冲。安全气囊和安全带共同作用，防止乘员直接撞上车内坚硬部件，从而避免或减轻伤害。
•   安全带：将人牢牢固定在座椅上。
•   安全气囊：在人与方向盘、仪表板等硬物之间形成缓冲气垫。
🚗 约束空间 (Keep them restrained within the safe "life space")
“Life space” 指的是碰撞后，乘员身体周围仍能保持相对完整、可存活的空间。
•   安全带：负责将乘员“按在座位上”，防止其飞出车外或过度前移。
•   安全气囊：配合安全带，进一步限制头部和上半身的剧烈晃动，将人约束在安全的生存空间内。
> 安全气囊和安全带共同作用，一方面防止车内乘员在碰撞中撞上方向盘、挡风玻璃等硬物，另一方面将乘员牢牢固定在座位上，使其保持在安全的生存空间内。

•   🔩 Side Impact Bars (侧撞梁/侧门防撞杆)
    安装在车门内部，用于在侧面碰撞时吸收和分散撞击能量，从而加强车身、减少车门变形，以保护车内乘员。
•   🛡️ Collapsible Steering Columns (可溃缩转向柱)
    在正面碰撞时，转向柱会按设计折叠或断裂，以避免方向盘直接冲击驾驶员胸部和头部，起到保护驾驶员的作用。
    
头枕能够在追尾碰撞中有效减少颈部“挥鞭伤”的发生。
•   Head Restraints: 指汽车座椅上的头枕。
•   Whiplash injuries: 即“挥鞭伤”，指追尾碰撞时，乘员头颈部因突然加速-减速而产生的剧烈甩动，所造成的颈部软组织损伤。
•   Minimize... during rear-end collisions: 表明在追尾这类事故中，头枕可以起到减小伤害风险或严重程度的作用。

主动安全系统重在“防患于未然”，被动安全系统重在“亡羊补牢”；未来，这两者与智能交通及自动驾驶技术将深度融合，共同致力于大幅降低交通事故率。
🛡️ 主动系统 vs. 被动系统
•   主动安全系统 (Active Systems)
    ◦   目标：避免事故发生。
    ◦   工作方式：通过传感器和控制系统，主动介入车辆操作。例如，在可能发生碰撞时自动刹车、保持车道或调整车速。
•   被动安全系统 (Passive Systems)
    ◦   目标：减轻事故后果。
    ◦   工作方式：在事故发生时或被撞后被动触发，通过物理方式保护乘员。例如，安全气囊、安全带和安全车身在碰撞时发挥作用，以减少伤害。
🚀 未来发展趋势
未来的发展方向是将主动/被动安全系统与智能交通系统 (ITS) 及自动驾驶功能深度结合。通过车与车、车与道路基础设施之间的信息互通，以及自动驾驶的精确控制，实现对潜在危险的更早识别、更优决策和更快响应，从而进一步降低交通事故的发生率。

汽车主动安全技术旨在通过智能系统预测并避免事故发生（防患于未然），如ABS、ESP、自动刹车（AEB）；被动安全技术则是在碰撞发生时保护乘员和行人，减少伤亡，如安全气囊、安全带、高强度车身结构。主动安全侧重“预警与干预”，被动安全侧重“减轻后果”。 

一、 汽车主动安全系统 (Active Safety)
主动安全装置作用于事故发生前，通过传感器和电子设备监测环境，保持车辆操控稳定性，从而降低事故发生概率。 
制动与稳定系统：ABS（防抱死制动系统）、EBD（电子制动力分配）、ESP/ESC（车身电子稳定系统）、TCS（牵引力控制系统）。
预警与辅助系统：FCW（前碰撞预警）、AEB（自动紧急制动）、LDW（车道偏离预警）、LKA（车道保持辅助）、BSM/BSD（盲点监测）、ACC（自适应巡航）。
辅助装置：TPMS（胎压监测）、倒车影像/全景影像。
二、 汽车被动安全系统 (Passive Safety)
被动安全装置在碰撞发生时工作，主要目的是将车辆内部空间安全保持，并通过材料变形吸收撞击能量，保护人员。 
约束系统：安全带（预紧、限力）、安全气囊（前、侧、气帘）、儿童安全座椅。
车身结构：笼式车身、溃缩式前/后纵梁、吸能区设计、防撞钢梁。
其它保护：可溃缩式转向柱、发动机下沉技术、安全玻璃、碰撞后燃油自动切断装置。
三、 主动安全与被动安全的区别
作用时间：主动安全在碰撞前（预防）；被动安全在碰撞时（减轻）。
性质与组成：主动安全多为电子/传感器技术；被动安全多为物理实体装置。
目的：主动安全为了“不撞”；被动安全为了“撞了不重伤”。 
总结：现代汽车发展趋向于将主被动安全结合，形成完整的安全金字塔，主动安全预防为主，被动安全防御为辅，二者共同构成全方位的行车保障。
